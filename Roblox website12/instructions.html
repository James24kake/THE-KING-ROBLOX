Setting Up the Background Image

Note: The website is configured to use a Roblox characters image as the background. Follow these steps to set it up:

  1. Make sure the images directory exists in the root of your project folder.
  2. Right-click on the Roblox characters image and select "Save Image As..."
  3. Save the image as roblox-background.jpg in the images directory.
  4. Open the index.html file to see the website with the new background.

If the image doesn't appear, check that:

  • The file is named exactly roblox-background.jpg
  • It's saved in the images folder at the root of your project
  • Your browser might be caching the old version - try hard refreshing (Ctrl+F5)
